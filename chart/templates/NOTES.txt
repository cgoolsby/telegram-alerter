telegram-alerter is deployed.

In-cluster endpoints (Service {{ include "telegram-alerter.fullname" . }}:{{ .Values.service.port }}):
  POST /send                     free-form message  (bearer auth)
  POST /webhook/alertmanager     Alertmanager/Grafana webhook (bearer auth)
  GET  /healthz                  probe
  GET  /metrics                  Prometheus metrics

{{- if not .Values.secret.create }}

Using existing Secret: {{ .Values.secret.existingSecret }}
Create it if you haven't:
  kubectl -n {{ .Release.Namespace }} create secret generic {{ .Values.secret.existingSecret }} \
    --from-literal=TELEGRAM_BOT_TOKEN=... \
    --from-literal=TELEGRAM_CHAT_ID=... \
    --from-literal=AUTH_TOKEN=...
{{- end }}

Send a test page from your laptop:
  kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "telegram-alerter.fullname" . }} 8080:{{ .Values.service.port }}
  curl -X POST http://localhost:8080/send -H "Authorization: Bearer <AUTH_TOKEN>" \
    -H "Content-Type: application/json" -d '{"message":"hello from helm"}'
